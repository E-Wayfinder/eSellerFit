IF SOMETHING BREAKS:
================================================================================

Error: "ESF_SHELL is not defined"
→ All JavaScript files loaded? Check browser console (F12)
→ Paths correct? Should be ./assets/js/calculator-shell.js

Error: "Cannot read properties of undefined"
→ Missing shared files? Make sure you copied config.js and calculator-shell.js

Legal links showing 404?
→ Create the /legal/ folder with 5 HTML files

Results not showing?
→ All 12 questions answered? Check form validation
→ Check browser console for errors
